Quel est l’objectif de ce projet ?

H2PiyR constituera un véritable banc de test à échelle réelle pour démontrer les avantages de l'utilisation des piles à combustible hydrogène pour véhicules électriques (pile à combustible Véhicules-FCEV).


En complément, il permettra de :

• créer un réseau de mobilité hydrogène entre les pays de l'Union Européenne,
• réduire les émissions associées aux véhicules traditionnels,
• stimuler les économies locales,
• promouvoir les opportunités d'affaires, en particulier pour les PME/PMI,
• promouvoir l'innovation dans les technologies de l'hydrogène dans tous les domaines de connaissances nécessaires pour mettre en œuvre le couloir.
